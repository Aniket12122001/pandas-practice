# 📊 Employee Data Analysis using Pandas

## 🎯 Objective
The objective of this project is to analyze employee data to extract meaningful insights for better workforce understanding and decision-making.

This analysis aims to:
- Analyze salary distribution across employees
- Compare departments based on salary and employee count
- Identify top-performing employees
- Understand city-wise employee distribution
- Generate insights using data analysis techniques

---

## 🛠️ Tools & Technologies
- Python
- Pandas
- Jupyter Notebook

---

## 📂 Dataset
The dataset contains employee information such as:
- Employee ID
- Name
- Department
- Salary
- Experience
- City

---

## 📊 Analysis Performed
- Total number of employees
- Average salary calculation
- Highest paid employee identification
- Department-wise salary analysis
- Employee distribution by department and city
- Filtering high salary and experienced employees
- Salary categorization (High, Medium, Low)
- Top 3 highest paid employees

---

## 🔑 Key Insights
- IT department shows higher average salary compared to others
- Certain cities have a higher concentration of employees
- A small group of employees contributes to the highest salary bracket
- Experience generally impacts salary levels
