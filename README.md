# 📊 Sales Data Analysis Project

## 🔹 Objective
This project analyzes sales data to:
- Calculate total revenue
- Identify top-performing products
- Analyze city-wise sales
- Understand monthly trends
- Study customer payment behavior

## 🔹 Tools Used
- Python
- Pandas
- Matplotlib

## 🔹 Key Insights
- Top city contributing to revenue: Delhi
- Most popular product: Laptop
- Peak sales month: 2024-01(January)
- Most used payment method: UPI

## 🔹 Project Structure
- data/ → sales_data
- notebook/ → analysis file

## 🔹 How to Run
1. Download dataset
2. Open Jupyter Notebook
3. Run all cells
